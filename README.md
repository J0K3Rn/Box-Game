# Box Game
A simple game created with HTML and JavaScript

Link: https://j0k3rn.github.io/Box-Game/ 
 
 Current Features:
 - A moving box with gravity
 - Diagonal movement
 - Restart and clicker button
 - Game borders that block you from going outside the play area
